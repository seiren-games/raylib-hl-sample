# raylib-hl-sample
raylib on hashlink

# How to use
In windows-os.

1. Download raylib, and put it in `hashlink-extension/lib`
1. Run vscode from vcvars64.bat: Double-click `start-vscode-from-vcvars64.bat`
1. Build and run: vscode menu - `Terminal` - `Run Build Task...` or Ctrl+Shift+B
1. rename
